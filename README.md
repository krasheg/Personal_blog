# Personal Blog

This is a Django-based personal blog where users can create, edit, and publish blog posts, as well as comment on existing posts. 

## Features

- Login as superuser to create, edit, and publish blog posts
- Posts can be saved as drafts before being published
- Users can format their posts by emphasizing text, making it bold, or changing its size (h1 or h2)
- Visitors can view published blog posts and leave comments
- Superusers can approve comments before they are displayed publicly

## Getting Started

### Prerequisites

- Python 3.6 or higher
- Django 3.1.3 or higher

### Installation

1. Clone this repository to your local machine: `git clone https://github.com/krasheg/Personal_blog.git`
2. Navigate to the cloned repository: `cd Personal_blog`
3. Install the required packages: `pip install -r requirements.txt`
4. Run the server: `python manage.py runserver`
5. Visit `http://localhost:8000` in your web browser to access the blog

## Usage

- Log in as superuser to create, edit, and publish blog posts
- Use the formatting options when writing a post to emphasize text, make it bold, or change its size
- Save posts as drafts before publishing them
- Visitors can view published blog posts and leave comments
- Superusers can approve comments before they are displayed publicly
